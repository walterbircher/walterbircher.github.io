Design Review

Five visual directions for the same cureEDMD site.

Each concept keeps the current content, language support, navigation, hero rotation, stories, counters, and donation interactions intact. What changes is the visual voice: palette, typography, spacing, card treatment, and some homepage layout rhythm.

How to use this

  • Open any concept from the homepage preview button.
  • Click around the site to see that concept carry across pages.
  • Use the base site link if your team wants to compare against the current version.
